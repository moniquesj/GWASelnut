Here, we store and share the scripts used for data analysis in Salardi-Jost, M., Mager, S., Dini, G. et al. (2025) (doi.org/10.1186/s12870-025-07596-2), BMC Plant Biology. The collection primarily consists of R scripts, with some routines executed in Bash. Detailed descriptions are provided within the scripts. The raw reads from the sequencing data used in this study can be accessed at the European Nucleotide Archive (ENA) under the accession number PRJEB82325.
